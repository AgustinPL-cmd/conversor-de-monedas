## 📌 Descripción

Este proyecto es un conversor de monedas en tiempo real que utiliza la API de ExchangeRate-API.  
Con esta aplicación, puedes convertir fácilmente entre diferentes monedas utilizando las tasas de cambio más recientes disponibles.

## 🚀 Características

- **Conversión en Tiempo Real**: Obtén tasas de cambio actualizadas en tiempo real.
- **Interfaz Intuitiva**: Una interfaz de usuario fácil de usar y atractiva.
- **Multiples Monedas**: Soporte para múltiples monedas globales.
- **Historial de Conversión**: Guarda y visualiza conversiones pasadas.
